### Color

```jsx
<Alert color="success">
  <strong>Well done!</strong> You successfully read this important alert
  message.
</Alert>
<Alert color="info">This is not super important.</Alert>
<Alert color="warning">Oh oh...</Alert>
<Alert color="danger">Oh snap!</Alert>
```

### Dismissible

```jsx
<Alert color="warning" dismissible>
  Hey... dismiss me.
</Alert>
```

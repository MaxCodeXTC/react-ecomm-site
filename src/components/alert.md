### Color

```jsx
<Alert color="success" className="mb-2">
  <strong>Well done!</strong> You successfully read this important alert
  message.
</Alert>
<Alert color="info" className="mb-2">This is not super important.</Alert>
<Alert color="warning" className="mb-2">Oh oh...</Alert>
<Alert color="danger" className="mb-2">Oh snap!</Alert>
```

### Dismissible

```jsx
<Alert color="warning" dismissible>
  Hey... dismiss me.
</Alert>
```
